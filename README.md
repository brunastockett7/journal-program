# Journal Program (C# Console App)

This is a simple journal application built in C# that helps users record daily thoughts.

## ✨ Features
- Random writing prompt generator
- Mood tracker for each entry
- Saves entries to a `.json` file
- Loads entries from a saved file
- Fully console-based UI with menu options

## 🛠 How to Run

1. Clone this repository  
2. Open a terminal in the project folder  
3. Run the program:

```bash
dotnet run
